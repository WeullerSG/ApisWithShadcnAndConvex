#  APIS com ShadCN e Convex

Bem-vindo ao projeto **APIs with ShadCN and Convex** — uma aplicação moderna que demonstra a integração entre **Next.js**, **Convex** e **ShadCN UI**.

**Acesse o projeto online:**  
https://apis-with-shadcn-and-convex-zzof.vercel.app/

---

## Sobre o Projeto

Este projeto foi desenvolvido para demonstrar:

- Criação de APIs usando **Convex** como backend realtime.
- Construção de interfaces limpas utilizando **ShadCN/UI**.
- Integração com o **Next.js** (App Router).
- CRUD completo com armazenamento real no Convex.
- Uso de mutations, queries e actions.

Serve como base de estudo, template ou referência para projetos que utilizem a stack moderna **Next.js + Convex + ShadCN**.

---

## Tecnologias Utilizadas

- Next.js 15  
- Convex  
- ShadCN/UI  
- Tailwind CSS  
- React Hook Form + Zod  
- TypeScript  
- Deploy na Vercel
